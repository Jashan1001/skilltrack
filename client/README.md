# SkillTrack Client

Frontend application for SkillTrack — a structured DSA learning platform with real-time code execution.

It provides a SaaS-style interface for navigating problems, writing code, and tracking progress.

---

## ⚙️ Role in System

The client handles:

- User interaction and navigation  
- Problem browsing and structured roadmap  
- Code editing and submission workflow  
- Displaying execution results and progress  

It communicates with the backend API, which in turn interacts with the execution engine.

---

## 🧱 Tech Stack

- React  
- TypeScript  
- Tailwind CSS  
- React Router  

---

## 🧩 Core Views

- **Dashboard** → Overview and quick navigation  
- **Patterns** → Guided DSA roadmap  
- **Pattern Detail** → Timeline-based learning  
- **Master Sheet** → Complete problem index  
- **Problem Detail** → Code editor (run/submit flow)  
- **Submissions** → Attempt history  
- **Leaderboard** → Rankings  
- **Admin Panel** → Problem management  

---

## 🎨 UI Features

- SaaS-style layout with sidebar navigation  
- Light / Dark theme support  
- Structured curriculum navigation  
- Minimal, distraction-free coding interface
