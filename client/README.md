# SkillTrack Client

The frontend provides a SaaS-style interface for structured DSA learning.

---

## 🧱 Tech Stack

- React
- TypeScript
- Tailwind CSS
- React Router

---

## 🧩 Core Views

- Dashboard
- Patterns (Guided Roadmap)
- Pattern Detail (Timeline)
- Master Sheet (Problem Index)
- Problem Detail (Editor)
- Submissions
- Leaderboard
- Admin Panel

---

## 🎨 UI Features

- Collapsible sidebar
- Light / Dark theme
- Hybrid roadmap model
- Structured curriculum sorting
- Clean SaaS layout

---

## 🚀 Run Client

```bash
npm install
npm run dev
```
