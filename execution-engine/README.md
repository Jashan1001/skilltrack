# SkillTrack Execution Engine

The execution engine runs user-submitted code securely inside Docker containers.

---

## 🐳 Purpose

- Isolate user code from main server
- Support multiple languages
- Run public & private test cases
- Enforce strict evaluation

---

## 🧱 Supported Languages

- JavaScript
- Python
- C++

---

## 🔐 Security Model

- Container isolation
- Timeouts
- Memory limits
- No network access inside containers

---

## 🚀 Run Execution Engine

```bash
npm install
npm run dev
```

Make sure Docker is installed and running.
